Japa 108 Watch

Standalone Apple Watch app — current build

Captured 2026-04-21 on Apple Watch Series 10 (46mm) / watchOS 11.2 simulator. Post Sprint 2 refactor: goal semantics in beads end-to-end, cream + dark face themes, Ring + Number display modes, sheet-overlay and picker-contrast UX fixes. All 66 tests pass.

Onboarding

1. Choose your display (first run)

Pick Ring (circular progress) or Number (large digit, VoiceOver-recommended). Remembered per-user. The "of 1008" hint reflects the bead-count refactor (5853c95).

2. Home — ready to start

Saffron Start button shows the current goal in beads. Gear opens Settings, "Change goal" jumps to the picker.

3. Goal picker (post-refactor)

Presets are 11 / 54 / 108 / 1008 / 10008 beads with subtitles ("Quick practice", "Half mala", "One mala", "Nine malas", "Mahā count"). Plus a Custom entry.

Counting — both display modes

4. Ring display (cream)

Bead chain wraps the dial; current count fills in saffron. "BEADS · 84 of 108" mid-mala. Mala-boundary haptic fires every 108 taps.

5. Number display (dark)

Big digit + "of 108" subtitle. Maximum legibility for low-light japa or VoiceOver users.

6. Session complete

Full ring lights up at goal. Total beads + minutes shown. Saffron "Done" returns to home; secondary action to log/share.

Settings — both themes

7. Settings — cream

Face theme + Display mode as explicit segment buttons. f3f8723 rebuilt this from a Form/Picker that rendered selected text invisibly white-on-cream.

8. Settings — dark

Same component, dark face. Display switched here to Number — selection is unambiguous in either theme.

9. Account & About

Sign out (saffron, destructive). Version row shows build identity for support.

What's in this build (vs Sprint 2 baseline)

Bead refactor (5853c95) — goal semantics switched from malas to beads end-to-end. Presets now [11, 54, 108, 1008, 10008]. All "MALA X OF Y" text replaced with "BEADS · 108 of 108" style. Pre-refactor mala values (1, 3, 11) migrate to 108.

UX fixes (f3f8723) — three watchOS-specific regressions resolved. (1) engine.start() now accepts .discarded as a valid entry state, not just .idle. (2) .sheet presentations replaced with NavigationStack pushes to skip the un-stylable system clock + X overlay. (3) Settings picker rebuilt as tinted segment buttons (no more white-on-cream selected value).

Dark face theme — full theme system. Same saffron accent works on both.

New app icon (5853c95) — saffron-gradient mala-bead artwork from the legacy iOS target. Seven watchOS sizes auto-generated via sips.
